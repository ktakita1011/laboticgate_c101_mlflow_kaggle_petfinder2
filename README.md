# らぼちっくげーとmlflowとPyTorch Lightningを利用したらくらく実験管理のリポジトリ
学習コードはこちらになります。
dockerの学習環境を立ち上げて実行する流れです。

docker-composeのバージョンはdocker-compose<=1.29.1

使い方
```
$docker-compose up
or
$docker-compose up -d
```

dockerコンテナが立ち上がるとjupyter server{http://localhost:8888}が立ち上がります。
パスワードはrun.shに記載してあります。
